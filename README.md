# IBM HR Employee Attrition Analysis

Advanced HR analytics project using SQL to uncover deep attrition drivers.

## Key Insights
- Overall attrition: 16.12%
- Overtime workers: 30.5% attrition (vs 10.4% no overtime)
- Underpaid by tenure bucket: highest risk in 0-2 years
- Top flight risks ranked per department using risk scoring
- Career stagnation drives ~28% attrition
